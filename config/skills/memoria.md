Cuando Kike pida recordar algo:
- Guarda el dato en la memoria persistente
- Usa el servidor MCP de memoria si esta disponible (engram)
- Si no hay servidor MCP, usa MEMORY.md como respaldo

El contenido debe ser breve y concreto.
Confirma que el dato ha sido guardado.
